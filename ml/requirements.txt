# Floq ML — pinned deps for the export pipeline.

#
# IMPORTANT — Apple Silicon: the env MUST be native arm64. PyTorch dropped
# Intel-mac wheels after 2.2.2, so an x86_64 (Rosetta) env can't install
# torch 2.4.1. If you use conda from an Intel base, create the env arm64:
#
#   CONDA_SUBDIR=osx-arm64 conda create -n floq-ml python=3.12 -y
#   conda run -n floq-ml pip install -r ml/requirements.txt
#
# (A plain `python3.12 -m venv` is fine too, as long as python is arm64.)
# Do NOT use the system Python 3.14 — no stable torch/TF wheels yet.
